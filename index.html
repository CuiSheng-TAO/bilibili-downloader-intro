项目背景

一个简单的个人需求，如何变成一个完整的开源工具？

起点：弹吉他的人需要一个好用的下载器

我喜欢弹电吉他，经常需要把 B 站上吉他手的现场演奏视频保存下来，导入库乐队 (GarageBand) 进行跟弹录音练习。市面上的下载工具要么界面复杂、要么需要付费、要么操作繁琐，于是决定自己做一个。

项目的下载核心借鉴了 GitHub 上的经典开源项目 yt-dlp，我在此基础上用 Python + Flask 封装了一套 Web 交互界面，做到粘贴链接即可预览和下载，双击启动脚本即用。

坦诚说明：本项目使用 Vibe Coding 方式开发，借助 Claude Code 辅助编码。下载引擎基于开源项目 yt-dlp，我负责需求定义、界面设计、功能整合和用户体验把控。

功能展示

每个功能都围绕"简单好用"的原则设计，让非技术用户也能轻松上手。

智能解析

  • 输入 B 站 URL 自动获取视频标题、封面、时长、UP主
  • 实时预览视频缩略图，确认后再下载
  • 支持多种 B 站链接格式（BV号、av号、短链接）

多格式下载

  • 视频支持 MP4 / WebM 格式
  • 音频支持 MP3 / M4A / FLAC 格式提取
  • 画质从 360p 到 4K (2160p) 全覆盖
  • 自动调用 FFmpeg 进行音视频合并

视频裁剪

  • 设置开始和结束时间点，只下载需要的片段
  • 通过 FFmpeg 精准裁剪，无需下载完整视频
  • 适合提取视频中的精彩片段或教程关键段落

用户体验细节

  • 下载历史侧边栏，快速回顾和访问已下载文件
  • 可选保存视频封面高清缩略图
  • 启动时自动检测 Python 依赖和 FFmpeg 环境
  • 下载路径可自定义配置，默认保存到桌面
